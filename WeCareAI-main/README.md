# 🏥 WeCare AI

An AI-powered healthcare assistance platform designed to simplify hospital navigation and improve patient experience through intelligent medical report analysis, conversational AI, and digital healthcare services.

## ✨ Features

- 🤖 AI-powered Medical Report Analysis
- 📄 OCR-based Text Extraction from Medical Reports
- 🏥 Smart OPD Department Recommendation
- 🎫 Digital Token Generation
- 💬 AI Healthcare Assistant Chatbot
- 🚨 Emergency Guidance & Patient Support
- 🌐 Responsive Web Interface
- 🌍 Multilingual Support (English & Hindi)

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/aadi121k/WeCare_AI.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

uvicorn main:app --reload
```
